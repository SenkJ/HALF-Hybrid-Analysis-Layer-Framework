# HALF - 网络数据包捕获和分析系统

HALF (Hybrid Analysis Layer Framework) 是一个功能完整的网络数据包捕获和分析系统，支持实时数据包捕获、多层协议解析、数据存储和可视化分析。

## 🚀 功能特性

### 核心功能
- **实时数据包捕获**: 使用libpcap库捕获网络数据包
- **多层协议解析**: 支持以太网、IP、TCP、UDP、ICMP、ARP等协议
- **实时数据展示**: Java Swing GUI界面实时显示捕获的数据包
- **数据持久化**: MySQL数据库存储捕获的数据包信息
- **统计分析**: Python脚本进行数据包统计和可视化分析

### 扩展功能
- **GUI集成**: 完整的图形用户界面，支持设备选择、流量控制等

## 🛠️ 技术架构

### 后端模块
- **C语言核心**: 数据包捕获和协议解析 (`half.c`)
- **数据库操作**: MySQL数据存储

### 前端界面
- **Java Swing**: 图形用户界面 (`half.java`)
- **JNI接口**: C语言和Java之间的桥接

### 数据分析
- **Python脚本**: 数据可视化和统计分析 (`GUItorch.py`)


## 🚀 快速开始

### 环境要求
- Linux操作系统
- GCC编译器
- Java 8+
- Python 3.6+
- MySQL数据库
- libpcap开发库

### 编译和运行

1. **编译原生库**:
```bash
cd NEW-HALF
./gcc.sh
```

2. **编译Java程序**:
```bash
cd src/HALF
javac half.java
```

3. **运行程序**:
```bash
java half
```

4. **数据分析**:
```bash
python3 src/GUItorch.py
```

## 🔧 配置说明

### 数据库配置
项目使用MySQL数据库存储数据包信息，需要提前创建数据库和表结构。

### 网络权限
数据包捕获需要root权限或相应的网络权限。
